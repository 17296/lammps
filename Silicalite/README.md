## Silicalite SiO2 MFI structure 

### Description

The simulation consists of SiO2. The force field is SiO.tersoff, a bond order potential.

![Algorithm schema](./SilicaliteTersoff.jpeg)

### How to

Run the input.lammps script using LAMMPS.
